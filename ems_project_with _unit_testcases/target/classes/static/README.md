# EMS Frontend Pages

All UI pages are ready to use. Simply open any HTML file in your browser.

## Available Pages

### User Pages
- **index.html** - User Login Page
- **register.html** - User Registration Page
- **recover-password.html** - Password Recovery/Reset Page
- **events.html** - Event Listing Page (All Events)
- **event-registration-individual.html** - Individual Event Registration
- **event-registration-group.html** - Group Event Registration
- **dashboard.html** - User Dashboard (My Registered Events)

### Admin Pages
- **admin-login.html** - Admin Login Page
- **admin-dashboard.html** - Admin Dashboard

## Quick Start

1. Navigate to the `frontend` folder
2. Open any HTML file in your web browser
3. All pages use Tailwind CSS from CDN (no local installation needed)
4. Pages are fully responsive and work on all devices

## Features

- Clean, modern UI design
- Responsive layouts for mobile, tablet, and desktop
- Tailwind CSS for styling
- Material Symbols for icons
- Ready for backend integration

## Notes

- All links between pages are already configured
- Each page has placeholder text and sample data for demonstration
- The UI is designed to be integrated with backend services
- Password visibility toggles are functional
- All forms are ready to be connected to API endpoints
